## Why AACommodate API?

This page is for developers who want to learn how to use our npm package to enable voice interaction in their apps.

While our primary focus has been AAC (Augmentative and Alternative Communication) game developers, the AACommodate API can be used by any developer looking to integrate voice-activated controls into their project with ease.

# How to add to your Project
